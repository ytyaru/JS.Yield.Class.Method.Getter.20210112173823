# Usage

1. Start the terminal
    1. Move the current directory. To the path where index.html is. `cd JS.Yield.Class.Method.Getter.20210112173823 / src`
    2. `python3 -m http.server 8000`
2. Launch your browser
    1. Enter `http://0.0.0.0:8000/` in the URL field
